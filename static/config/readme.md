# 配置文件说明

#### notice.json页面通知配置
```
{
 "notices": [
    {
      "title": "通知1",          //标题
      "content": "通知1的内容",  // 内容
      "type": "warning",        //类型： success,info,warning,error
      "duration": 10000         //持续时间，0：表示不会关闭
    },
    {
      "title": "通知",
      "content": "报表平台将于19:00重启升级，预计耗时15分钟左右。",
      "type": "warning",
      "duration": 10000
    }
 ]
}
```
